Plateforme clinique IA

Aide au diagnostic pulmonaire, rapide et interpretable.

PulmonaryAI analyse les radiographies thoraciques pour estimer les pathologies probables, avec visualisation de confiance et export API pour vos workflows.

Cette plateforme est concue pour l'usage clinique assiste: un operateur charge l'image, le modele calcule les probabilites, puis le resultat est enregistre avec les metadonnees patient pour audit et suivi.

Comment utiliser la plateforme

Etape 1

Remplir les infos patient (nom, age, genre, date examen).

Etape 2

Uploader une radiographie thoracique (JPG, JPEG, PNG).

Etape 3

Lire la classe principale et les probabilites detaillees.

Etape 4

Consulter l'historique pour comparaison, suivi et tracabilite.

Interpretation recommandee

Le score de confiance aide a prioriser les cas, mais ne remplace pas l'expertise medicale. Utilisez le resultat comme outil d'aide a la decision.

Pourquoi ces techniques

  • Django: gestion fiable des vues, formulaires, sessions et base de donnees.
  • TensorFlow/Keras: inference CNN pour classification multi-classes en imagerie thoracique.
  • SQLite (dev): demarrage rapide sans infra supplementaire.
  • API REST: integration simple avec autres apps, outils labo ou mobile.
